Web Analytics System

Our own robust constellation of Google Cloud Platform services,
dashboards, pipelines, predictive modeling, and analytics to get insights
into user behavior on websites and applications.

Cloud Native System

Argot Nova's Web Analytics System is built using Google Cloud Platform component services, but delivered as an off-the shelf product through infrastructure-as-code deployment. Our templated service includes storage, analytics, monitoring, dashboarding, alerts, anomaly-detection, and behavior modeling, resulting in a system that can be easily deployed to gather and analyze user activity for up to thousands of websites and web applications without the need for the usual months or years of time required to build and configure a cloud infrastructure and analysis posture.

Capabilities
  • Monitor referrals, clicks, downloads, conversions, media interactions
  • Storage and anomaly-detection for years of user behavior data
  • Automatic migration to cold storage and cost savings
  • Anomaly-detection and removal of non-human traffic from analysis
  • Built-in translation to and from dozens of languages configurable in analytical pipelines
  • Included web analytics monitoring dashboard with easy connectivity to BI tools like Looker, Tableau, and PowerBI
  • Sensitive data masking
  • User and Admin group access controls able to be configured for each organization
